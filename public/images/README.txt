# public/images/

Place your portrait photos here named exactly:

  frank1.png
  frank2.png
  frank3.png
  ...
  frank20.png

- frank1.png → used on the About page hero section
- frank2.png → used on the Home hero + Philosophy page
- frank3.png → frank20.png → used in the infinite gallery on the About page

If a gallery image is missing it will silently fall back to frank1.png or frank2.png.

Also place in /public/:
  og-image.jpg          (1200 × 630px — for social sharing previews)
  favicon.ico
  favicon-16x16.png
  favicon-32x32.png
  apple-touch-icon.png  (180 × 180px)
  android-chrome-192x192.png
  android-chrome-512x512.png
